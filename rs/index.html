Getting Started


Windows + Linux Users Only

1. Download the D3DX9_42.dll file and place it in the main directory of the game.
2. When downloading CDLC (AKA: Custom DLC) it will always be a .psarc file, always save this in the "dlc" directory in your game folder
3. After every dlc installation you have to restart your game to load it in

MAC users only


1. Instead of downloading the D3DX9.dll file please visit here
2. When downloading CDLC (AKA: Custom DLC) it will always be a .psarc file, always save this in the "dlc" directory in your game folder
3. After every dlc installation you have to restart your game to load it in
